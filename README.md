# Customer Affiliate App

This is a Django web application that allows managers to create, edit, and manage customers, and track their eligibility for cash-out.

---

## 💼 Features

- 👤 Manager login & dashboard
- ➕ Create / Edit / Delete customers
- ✅ Mark customers eligible for cash-out
- 💳 Add credit card details for payout
- 📆 Generate payout date (1 month ahead)
- 🔍 Search customers by name, email, or phone
- 👀 Customers can log in and see their balance, payout status, and credit card info

---

## 🛠 Tech Stack

- **Backend**: Django 5.x
- **Frontend**: HTML5 + Bootstrap 5
- **Database**: SQLite (for development)
